📋 簡介 Briefing

本網站是一套以 Flask 為後端的台鐵時刻表查詢系統，資料來源為交通部 TDX 運輸資料流通服務平臺。具備深色主題 RWD 介面、多層快取（瀏覽器 → 伺服器記憶體 → 磁碟），並部署於 Render.com。

🚀 功能 Features

  • 常態班表查詢 — 任意兩站間靜態時刻表查詢
  • 當日班次查詢 — 指定日期實際行駛班次
  • 即時狀態 — TDX LiveBoard API 即時誤點疊加
  • 車站即時看板 — 出發/到達站即時到離站看板（往北/往南）
  • 列車詳細頁 — 單一車次停靠站清單、即時位置、通阻通知、最新消息
  • 車種篩選 — 對號列車 / 區間車 / 區間快
  • 常用車站 — 自動追蹤常用站
  • 城市分組選站 — 依城市分組並標示車站等級（◆ ● ★ ▸）
  • 顯示筆數控制 — 5 / 7 / 8 / 9 / 10 / 12 / 15 / 20 / 全部
  • Mobile 響應式 — 針對 ≤600px / ≤768px 斷點優化

🔒 安全機制 Security

措施實作
Content Security Policy@app.after_request — default-src 'self', frame-ancestors 'none'
X-Content-Type-Optionsnosniff
X-Frame-OptionsDENY
XSS 防護兩個 JS 檔案中的 escHtml()；URL 參數使用 encodeURIComponent()
輸入驗證車站代碼比對 _VALID_CODES；車次號碼：^\d{1,5}$；日期：嚴格 YYYY-MM-DD + 範圍檢查
Debug 端點保護/debug/cache 非 debug 模式回傳 404
原子性磁碟寫入tmp 檔案 + Path.replace()（seed 與 timetable cache 皆使用）
執行緒安全_cache_lock, _pool_lock, _station_load_lock, _save_lock

鎖保護對象

_station_load_lock_station_load_done（確保一次性車站 API 載入）
